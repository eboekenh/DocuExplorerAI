## Quick Start Guide

### Welcome to DocuWeb! 🎉

DocuWeb transforms how you interact with documents by adding intelligent, interactive annotations powered by AI. Upload any document and start asking questions to get instant explanations, summaries, and insights.

### 30-Second Setup
1. **Visit**: [https://docuweb-app.vercel.app](https://docuweb-app.vercel.app)
2. **Sign Up**: Create your free account with email and password
3. **Upload**: Drop your first document or paste text content
4. **Select & Ask**: Highlight any text and ask your question
5. **Get Answers**: Receive AI-powered explanations instantly

### What You Can Do
- 📄 **Upload Documents**: Support for Word, PDF, and text files
- 🤖 **AI Questions**: Ask anything about your document content
- 💬 **Smart Annotations**: Get explanations, summaries, and analysis
- 🌍 **Multi-Language**: Interface available in English, Turkish, and German
- 💾 **Auto-Save**: All your work is automatically saved
- 🔄 **Session Management**: Organize and revisit your document sessions

## Core Features

### Document Upload and Management

#### Uploading Documents

**Method 1: File Upload**
1. Click **"Upload Document"** button
2. Select file from your computer
3. Choose from supported formats: Microsoft Word (.docx, .doc), PDF files (.pdf), Plain text (.txt)
4. Wait for processing and text extraction
5. Review the imported content

**Method 2: Copy and Paste**
1. Click **"Paste Text"** option
2. Paste your content directly into the text area
3. Add a document title
4. Click **"Create Session"**
